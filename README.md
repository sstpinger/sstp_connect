# sstp_connect

sstp_connect is a plugin to connect MS-SSTP servers, like Softether

Inspired by [https://github.com/kittoku/Open-SSTP-Client]

Open to contribution

### Supported platforms for now
| Platforms | Support |
|-----------|:-------:|
| android   |  yes    |
| ios       |  no     |
| linux     |  no     |
| windows   |  no     |
